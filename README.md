想象力比较丰富的朋友们可能钟爱读小说时脑海中浮现出的场景更胜于看电影动漫，但是每次都需要点击才能翻页可能又略有些麻烦而且稍微打断了沉浸感，尤其阅读速度稍微快一些就得翻页很多次，是否存在一种方法可以像看电影那样我只需要点播放键接下来两三小时就纯观赏就行了不需要点来点去呢？好消息，存在的，那也就是速读。但是找遍了市面上全部的速读工具都无法同时兼顾下述所有条目，所以自己做了一个。

1.需要能很容易的调整阅读速度，包括ctrl+↑↓长按可以调整阅读速度

2.市面上产品往往限定人的速读速度在400到1500以内原因不明，但个人需要它能达到3500以上

3.绝大多数的速读工具不支持中文或是支持有限，尤其是涉及小说阅读，多源头下载的txt小说文件里中文格式种类非常繁多

4.bookmark功能

5.改变字体大小

6.鼠标中键翻滚、nextpage翻页、拉动进度条调整阅读进度

做出来之后发现效果不错，于是沿用其设计在这基础上做了一个打字游戏版本，点击右侧的红字按钮就可以进入打字模式了！

由于市面上几乎不存在搜狗双拼输入法的双拼打字游戏，于是自己设计了一款。

一次输入练习模式：该模式下，以往双拼输入要两个按键配合，但在纯粹练习手速和精度时可能想要快一点找手感慢慢做减法，这个模式允许你这么做。

简单乱打模式：在打字模式下点击左下角的【简】进入简单模式，乱打就可以了，有时候你只是想找个打字的大致感觉实质上只是在看小说而已，这个模式就很好的做到了这点。

一起来快乐打字一边享受看小说吧~~ 打累了就换成小说速读~~
